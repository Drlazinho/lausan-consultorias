# SPA Site de Lausan.

Um projeto freela, criar um template de um site tipo SPA.
O design é simples, pois essa foi a ideia fornecida pelo cliente.

## Desenvolvimento
* Reactjs
* Styled-components
* React-Icons
* Swiper.js
* Responsivo Mobile/Desktop
